#AutoParts - Приложение для магазина автозапчастей

Мобильное приложение для управления каталогом автозапчастей, корзиной покупок и заказами.

Требования

Системные требования
- Android Studio: Arctic Fox или новее (рекомендуется последняя версия)
- JDK: версия 17 или выше
- Android SDK:
  - Минимальная версия SDK: 24 (Android 7.0 Nougat)
  - Целевая версия SDK: 34 (Android 14)
  - Компиляция SDK: 34
- Gradle: версия 8.13.2 (включена в проект)
- Kotlin: версия 2.0.21

Аппаратные требования
- Устройство: Android 7.0 (API 24) или выше
- Эмулятор: Android 7.0 или выше (для тестирования)
