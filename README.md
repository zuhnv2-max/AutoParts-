Мобильное приложение для бронирования авиабилетов

Архитектура:
Клиент: Android
Язык: Kotlin
База данных: Room Database, Sqlite(внутреняя) / Интеграция с backend
Требования: Android SDK 24+, Android Studio Arctic Fox и выше
MVVM Model-View-ViewModel - архитектурный паттерн

Установка:
Клонировать репозиторий
Открыть проект в Android Studio
Дождаться синхронизации Gradle
Запустить на эмуляторе или физ.устройстве
